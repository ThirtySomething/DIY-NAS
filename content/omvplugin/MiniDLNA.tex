\subsection{MiniDLNA}

To stream music and videos from the \gls{NAS} to the network I'm using the
\href{https://sourceforge.net/projects/minidlna/}{ReadyDLNA} media server
software. In \gls{OMV} this is used with the old name \tsTextMonospace{MiniDLNA}.

\tsImageO{./images/OMV-Plugin-MiniDLNA.png}{The MiniDLNA plugin}{\custCopyright{}}{width=\textwidth}

The basic setup is simple - I've checked the \tsTextMonospace{Enable} box and that's
it.

\tsImageO{./images/OMV-MiniDLNA-Settings.png}{The MiniDLNA settings}{\custCopyright{}}{width=\textwidth}

Then I have to define the shares and the kind of content of the shares.

\tsImageO{./images/OMV-MiniDLNA-Shares.png}{The MiniDLNA shares}{\custCopyright{}}{width=\textwidth}
